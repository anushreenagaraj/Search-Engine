# 🎨 Mood, Emotion & Sentiment-Based Image Search Engine

This project is an AI-powered image search engine where users can search for images based on mood, emotion, and sentiment — either by uploading an image or typing a natural language query.

## 🚀 Features

- 🔍 Search with text like "happy forest" or "sad city"
- 🖼️ Upload an image to detect mood, face emotion, and scene
- 🎯 Finds the best-matching images using CLIP similarity
- 🧠 Hybrid analysis using DeepFace, OCR, and CLIP
- 🖥️ Clean UI with Streamlit
